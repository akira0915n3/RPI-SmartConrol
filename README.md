# RPI-SmartConrol
RaspberryPi Zero 用赤外線読み取り、赤外線信号の発信シールドです。<br>
SMD部品を使用しているため、制作難易度は高めです。<br>
部品は秋月電子ですべて揃うはずです。


- Pin配置

|PIN No.|Sources|
|-------|-------|
|Pin 1  |3.3V   |
|Pin 2  |5V     |
|Pin 6  |GND    |
|GPIO 17|LED    |
|GPIO 18|LEDRead|
|Pin 17 |3.3V   |
|Pin 30 |GND    |
|GPIO 12|CN1    |
